# Snake Water Gun Game (Python OOP + Tkinter)

A desktop-based Snake Water Gun game built using **Python**, following **OOP principles** with a **Tkinter GUI**.

##  Features
- OOP-based backend logic
- Tkinter GUI frontend
- Score tracking
- Reset game option
- Sound effects
- Round history log

## Technologies Used
- Python
- Tkinter
- Object-Oriented Programming (OOP)

## How to Run
1. Clone the repository
2. Make sure Python is installed
3. Run the GUI file:

```bash
python gui.py
